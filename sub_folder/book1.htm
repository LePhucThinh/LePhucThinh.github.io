Python for Everybody: Exploring Data Using Python 3
Description: The book is suitable for everybody who wants to explore Python language without any other programming background. It provides a comprehensive and clear explanation on the fundamental concepts of Python such as data types, data structures, functions, conditional statements and loops, files, etc... through practical examples, diagrams, links, and exercises. But it missed to cover list comprehensions and lambda functions. The book covers the topics like regular expressions, databases, web services and visualization which are useful for who want to become a data analyst. But it does not cover the commonly used Python libraries like Pandas, NumPy, Matplotlib, etc.
